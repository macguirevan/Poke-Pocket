## Getting Started

### Prerequisites

Ensure you have the following installed before running the project:

- [Node.js](https://nodejs.org/) (Recommended: Latest LTS version)
- npm (Comes with Node.js)

### Install Dependencies

```
npm install
```

### Run Development Server

```
npm run dev
```
